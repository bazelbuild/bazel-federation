# Hello world in all the languages


## Things to try

```
bazel build :*
bazel run :hello_in_c
bazel run :HelloInJava
bazel run :hello_in_go
bazel run :hello_in_python
bazel run :hello_in_rust
bazel build :tarball
tar tvf ../bazel-bin/hello_bazel/tarball.tar
```
